# 云猫娘养成 

一个功能完整的 AstrBot 插件，包含云养猫娘养成、独立的经济系统。
📕 本项目参考zerobot中的云养猫并通过ai进行重构和优化

## ✨ 功能特色

- 🐱 **猫娘养成**：许愿、喂养、互动、打工、自定义形象
- 🎁 **签到系统**：每日签到获取宝石 内置了横板和竖版两套好看的签到卡片渲染以及文字输出
- 💰 **经济系统**：钱包、打工赚钱、转账交易
- 📊 **排行榜**：钱包排行、猫娘羁绊排行
- 🎨 **高度可定制**：自定义签到背景、自定义猫娘图片、自定义每日一言语录
- 🧩 **插件拓展页**：在 AstrBot 插件 Pages 中实时调整经济、许愿、状态结算、喂食、打工地点、互动和性格效果

## 📥 安装

1. 在 AstrBot 插件市场搜索「astrbot_plugin_neko_care」、「猫娘养成」
2. 点击安装并启用
3. 插件会自动创建必要的目录和默认资源

## 🎮 使用指令

### 签到与经济
- `签到` / `猫猫签到` - 每日签到领取宝石
- `查看猫猫钱包` - 查看余额
- `每日打工` - 主人每日打工赚取宝石
- `钱包转账 数量 @用户` - 转账给其他用户
- `钱包排行榜` - 查看财富榜

### 猫娘养成
- `请赐我一只可爱猫娘吧` - 唯一获取猫娘方式（默认80% 概率，3 天保底）
- `成长档案` / `猫娘状态` / `猫猫状态` - 查看猫娘成长档案
- `喂猫` / `喂猫娘` - 喂养猫娘（饱食度低于 85 时可喂；饱食度约 48 小时从 100 降到 0）
- `猫娘打工` / `猫猫打工` / `打工` - 让猫娘去耗时打工，消耗精力并在完成后领取宝石和成长值；可用 `猫娘打工 地点名` 指定地点，`猫娘打工 列表` 查看地点
- `撸猫` / `逗猫` / `摸猫` / `rua猫` - 消耗精力进行互动，增加心情、亲密度和成长值；有冷却和每日软上限，超出后收益递减
- `猫娘改名 名字` - 给猫娘改名
- `更换猫娘形象` + 图片 - 更换猫娘外观
- `羁绊排行榜` / `猫娘排行榜` - 查看本群猫娘羁绊排行

### 帮助
- `猫猫帮助` / `猫娘帮助` - 查看完整帮助

## ⚙️ 配置说明

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `coin_name` | 货币名称 | 宝石 |
| `sign_mode` | 签到模式（文字签到/图片签到） | 图片签到 |
| `keyword_trigger_enabled` | 是否允许直接关键词触发 | true |
| `sign_min_reward` | 签到最小奖励 | 80 |
| `sign_max_reward` | 签到最大奖励 | 150 |
| `work_min_reward` | 打工最小奖励 | 50 |
| `work_max_reward` | 打工最大奖励 | 120 |
| `catgirl_wish_probability` | 许愿成功概率 | 0.8 (80%) |
| `catgirl_wish_pity` | 许愿保底次数 | 3 |
| `appearance_change_price` | 形象券价格 | 1200 |
| `extra_admin_ids` | 额外管理员 ID 列表 | [] |

### 插件 Pages 配置

安装后可在 AstrBot 后台的插件拓展页打开 `neko-admin` 页面。页面会保存运行时配置到 `data/plugin_data/astrbot_plugin_neko_care/runtime_config.json`，保存后立即应用到当前插件实例。

可在页面中调整：

- 货币名、签到奖励、每日打工奖励和每日打工事件文本
- 许愿概率、保底次数、猫娘形象更换价格
- 饱食度、心情、健康、精力的按分钟线性结算参数和健康判定阈值
- 喂食阈值、低健康喂食倍率、食物池、食物价格和喂食收益
- 猫娘打工地点、耗时、奖励、阶段和精力档位收益倍率、消耗、成长和亲密收益
- 互动动作、互动文本、阶段门槛、冷却、精力消耗、每日软上限、心情收益倍率和递减倍率
- 8 种猫娘性格的倍率效果，包括状态消耗/恢复、喂食收益、打工收益/消耗、互动收益/消耗

新增互动动作可通过 `猫娘互动 动作名` 或 `猫猫互动 动作名` 触发；内置的 `撸猫`、`逗猫` 等短命令仍保留。

## 🐱 养成体系

猫娘养成以「羁绊」为核心，不再鼓励单纯追求体重数值。喂食、互动、猫娘打工会提升不同属性：

| 属性 | 说明 |
|------|------|
| `成长进度` | 由隐藏成长值换算为当前阶段百分比，推动猫娘从初遇、熟悉、亲近一路成长到永伴 |
| `亲密等级` | 由隐藏亲密度换算为等级，等级越高升级所需亲密度越多 |
| `饱食度` | 按分钟线性下降，过低会影响健康，归零超过 24 小时会离家出走 |
| `心情` | 按分钟线性下降，会受互动和打工影响；低心情会降低互动的成长/亲密收益，高心情会提高收益 |
| `健康` | 按分钟结算，长期饥饿或低心情会下降，照顾良好会缓慢恢复；低健康会降低喂食的附加收益 |
| `精力` | 按分钟线性恢复，猫娘打工和互动会消耗精力，高精力会提高打工报酬档位 |

成长阶段包括：初遇、熟悉、亲近、信赖、羁绊、灵契、永伴。喂食、互动、打工获得的成长值和亲密度仍按具体数值显示，状态档案中展示为成长进度和亲密等级。新猫娘初始体重为 40-90 斤范围内的小数存档，输出当前状态时显示整数，变动数值显示 1 位小数。旧版本的猫娘数据会在读取时自动补齐新字段，并将旧 kg 体重自动迁移为斤。

插件安装后会在 `data/plugins/astrbot_plugin_neko_care/assets/` 创建以下目录：

- `sign_backgrounds/` - 签到背景
- `catgirl_pool/` - 猫娘默认形象库
- `quotes.txt` - 一言语录文件

您可以添加自己的图片和语录，插件会随机选择使用。

## 📝 管理员指令

需在配置中添加 `extra_admin_ids`（AstrBot 自身管理员无需添加）：

- `管理员给 数量 @用户` - 给用户添加宝石
- `管理员扣 数量 @用户` - 扣除用户宝石
- `管理员查看 @用户` - 查看用户余额

## 🔧 常见问题


**Q: 如何更换签到背景？**  
A: 将图片放入对应目录，支持 jpg、png、webp 格式。

**Q: 猫娘数据是否群隔离？**  
A: 猫娘数据全局通用，但可通过「迁移猫娘到本群」登记到指定群，在该群羁绊排行榜中显示。
